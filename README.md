module.base 
===========

Structure and examples to create a new module.
Files and field to rename according to your project.

- [ ] Readme.md
- [ ] package.json
  - name
  - description
  - repository/url
  - bugs
  - homepage
- [ ] typings
  - mykoop-module directory
  - mykoop-module.d.ts
    - mkmodule
- [ ] lib
  - module.ts
    - mkmodule.Module

Components and style can stay like this but will have no effect until used
